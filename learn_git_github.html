Git
里面可用Linux命令 cd ll cat mkdir 等
设置名字和email
Git config --global user.name “KettyBety”
Git config --global user.email “gggh@qq.com”
仓库地址形式 https://github.com/KettyBety/hello-world.git

提交到github流程
1. 在相应的目录下克隆github仓库到本机 git clone https://github.com/KettyBety/hello-world.git
2. 进入下载的仓库 cd hello-world
3. 修改仓库内容后添加 git add .
4. 先提交到本地 git commit Cm “changes log”
5. 提交到github git push Cu origin master 要等待输入账号 密码

Github
添加新仓库 在页面上点击+号 写上仓库名 并初始化
删除仓库 点击Setting 在最下面确认删除
查找别人仓库 可在地址上输入 也可在搜索中写上仓库名按回车键
复制别人的仓库 点击Fork
